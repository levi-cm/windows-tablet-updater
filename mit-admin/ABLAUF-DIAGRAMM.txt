Wichtige Trennung
-----------------

Das sichtbare Fenster ist nur der Hinweis fuer Menschen am Tablet.
Die eigentliche Update-Installation laeuft ueber die Windows-Aufgabe
mit Systemrechten. Dadurch kann Windows Updates installieren, ohne dass
ein normaler Benutzer alles manuell bestaetigen muss.


Was wird aktualisiert?
----------------------

+--------------------------------------------------+
| Speicherbereinigung                              |
| - Temp-Dateien                                   |
| - Browser-Cache                                  |
| - WebView2-Cache                                 |
| - Windows Thumbnail-/Icon-Cache                  |
| - Papierkorb                                     |
| - Windows Update Downloadcache                   |
| - alte CBS-/DISM-Protokolle                      |
+--------------------------------------------------+
                         |
                         v
+--------------------------------------------------+
| Firefox                                          |
| - nur Mozilla Firefox                            |
| - kein "winget upgrade --all"                    |
+--------------------------------------------------+
                         |
                         v
+--------------------------------------------------+
| Windows Update                                   |
| - normale automatische Updates                   |
| - automatisch angebotene Treiber/Firmware        |
| - keine optionalen Preview-/Extra-Updates        |
+--------------------------------------------------+
